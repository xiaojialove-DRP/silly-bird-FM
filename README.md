# silly bird FM 🐦

一个朋友之间的声音电台。

低门槛：像发语音一样，人人给自己起个频道名（老电台频率的感觉），上传任何和声音有关的东西——一段话、一个故事、自己哼的歌、一场雨——分享给朋友。主打情感链接（听见朋友真实、有体温的声音），不是音质/制作。

一只常驻屏幕角落的小鸟，点开它就是一台可以在朋友的频道间来回切台的小电台，陪你 vibe-coding 时不再是一个人。

## 视觉参考

安徒生的剪纸艺术——奶油色纸片、单一底色、镂空的心形与花边、手工的毛边。整台机器做成一张会响的剪纸：小鸟站在深色的夜窗里，嘴边有剪纸的啾啾声波线。

## 现状

Web 原型阶段。

- **播放器**：小鸟收起停在桌面一角，点开弹出播放器，标题栏可拖动到屏幕任意位置，随时收回。（当前用网页模拟，最终会用 Tauri 包成真正常驻桌面的无边框小窗。）
- **真实播放**：把音频文件拖到小鸟身上（或点一下选择文件）即可播放，歌名 / 歌手 / 封面会自动从文件的 ID3 标签读取。
- **频道**：demo 频道作为占位展示调台的样子；真正「朋友互相上传、彼此切台收听」需要云端存储，是下一步。

## 路线

1. **播放器先行**（当前阶段）—— 小鸟窗口 + 剪纸美学 + 真实本地音频播放
2. **专辑管理** —— 给自己的频道 / 唱片取名，支持保存多张
3. **云端分享** —— 上传到云端存储，朋友之间真正能互相切台收听彼此的声音
4. **桌面化** —— 用 Tauri 包装，成为常驻桌面、可跨应用置顶拖动的小鸟
